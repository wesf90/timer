How to Use

Set Time: Enter seconds manually or use the +5s, +10s, +30s, +60s buttons to add time quickly.

Start Timer: Runs once and stops at zero (e.g., set 60s for a one-minute countdown).

Start Interval: Repeats automatically with a counter (e.g., set 30s for a constant timer that auto-restarts—great for workouts or reminders).

Stop: Ends the timer and resets the display.
